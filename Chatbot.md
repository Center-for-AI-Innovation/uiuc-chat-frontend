Chatbot.md

A Chatbot:

- belongs to a project
  - name, description, greeting, question, logo
  - materials, document group (?)
- LLM providers
  - API keys for LLM provider
- Tools
  - n8n tool & workflow
- API

A Chatbot can be accessed by

- Human users
  - Owner & admin of chatbot
  - Login users
  - Guest (?)
- Program
  - API

A User can interact with chatbot by

- Start or continue a conversation with a chatbot
  - manage conversation
    - delete conversation
    - edit conversation title
    - update model, document group, and tools
- Manage user interaction with chatbot
  - export conversation history
  - share chatbot: access control - private, public, log-in user only
    - email
- Config chatbot
  - Select X LLM providers, define default LLM
  - Upload materials from multiple sources
  - Update branding info
  - Update prompting and prompt setting

Materials

- Upload material
- Create document group
